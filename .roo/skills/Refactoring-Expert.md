# Skill: Clean Code & Refactoring Expert

## 🎯 목적
복잡하고 읽기 어려운 자바스크립트 코드를 2026년 최신 클린 코드 표준에 맞춰 정제한다. 초급 개발자도 이해할 수 있도록 가독성과 주석을 강조한다.

## 📏 핵심 규칙 (Core Rules)
1. **함수 10줄 제한**: 하나의 함수가 10줄을 넘어가면 논리 단위를 분리하여 서브 함수로 추출한다.
2. **의미 있는 명명**: `temp`, `data`, `obj` 같은 모호한 이름은 절대 금지. `isTodoCompleted`, `filteredTaskQueue` 등 목적을 명시한다.
3. **매직 넘버 금지**: 모든 고정 수치나 문자열 상수는 파일 상단에 `const CONFIG = {...}` 형태로 정의한다.
4. **JSDoc 필수**: 모든 함수 상단에는 매개변수와 리턴값의 타입을 설명하는 JSDoc 주석을 단다. 주석은 코드의 목적과 작동 방식을 명확히 설명해야 한다.

## 🚀 실행 프로세스
- 사용자가 "코드 정리해줘"라고 요청하면, 먼저 전체 구조를 스캔한다.
- 위 규칙을 위반한 항목을 리스트업한다.
- 수정 전/후의 가독성 점수를 비교하여 보고한 뒤, 실제 수정을 진행한다.
