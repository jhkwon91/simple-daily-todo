# 심플 데일리 투두 애플리케이션

## 프로젝트 개요

HTML, CSS, JavaScript로 구축된 간단하지만 기능이 풍부한 일일 투두 애플리케이션입니다. 이 클라이언트 측 웹 애플리케이션을 사용하면 카테고리, 우선순위, 마감일 및 진행 상황 추적 기능으로 사용자의 일일 작업을 관리할 수 있습니다.

## 주요 기능

- **작업 관리**: 할 일 항목 추가, 편집 및 삭제
- **분류**: 작업을 카테고리로 정리 (업무, 개인, 공부)
- **완료 추적**: 시각적 피드백으로 작업 완료 상태 표시
- **검색 기능**: 특정 작업을 빠르게 찾기
- **진행 상황 추적**: 완료 퍼센트를 표시하는 시각적 진행률 바
- **데이터 지속성**: 브라우저 세션 간에 localStorage를 사용하여 할 일 저장
- **데이터 내보내기/가져오기**: JSON으로 할 일 데이터 내보내기 및 가져오기
- **필터링**: 모든, 진행중, 완료된 작업 보기
- **반응형 디자인**: 모바일, 태블릿, 데스크탑 장치에서 작동
- **우선순위 수준**: 작업에 낮음, 중간, 높음 우선순위 할당
- **마감일 관리**: 작업 마감일 설정 및 추적

## 실행 방법

이 애플리케이션은 모든 최신 브라우저에서 직접 작동하는 순수 클라이언트 측 웹 애플리케이션입니다:

1. **직접 실행 방법**:
   - 브라우저에서 `simple-daily-todo/index.html`을 직접 열기

2. **로컬 서버 사용 (권장)**:
   - 터미널에서 프로젝트 디렉토리로 이동
   - 실행: `python3 -m http.server 8000` (Python 3) 또는 `python -m SimpleHTTPServer 8000` (Python 2)
   - `http://localhost:8000/simple-daily-todo/index.html` 방문

## 기술 세부사항

- **프론트엔드 기술**: HTML5, CSS3, JavaScript (ES6+)
- **저장소**: 데이터 지속성을 위한 localStorage
- **백엔드 필요 없음**: 순수 클라이언트 측 애플리케이션
- **반응형**: 모바일 우선 디자인 접근 방식
- **외부 의존성 없음**: 자체 포함 애플리케이션

## 사용 방법

1. 입력 필드에 작업을 입력하고 카테고리를 선택하세요
2. "추가" (Add) 버튼을 클릭하거나 Enter 키를 눌러 작업을 추가하세요
3. 체크박스를 사용하여 작업 완료 상태를 표시하세요
4. "수정" (Edit) 버튼을 사용하여 작업을 수정하세요
5. "삭제" (Delete) 버튼을 사용하여 작업을 제거하세요
6. "전체", "진행중", "완료" 버튼을 사용하여 작업을 필터링하세요
7. 검색 입력 필드를 사용하여 작업을 검색하세요
8. "데이터 내보내기" 버튼을 사용하여 데이터를 내보내세요
9. "데이터 가져오기" 버튼을 사용하여 데이터를 가져오세요

## 브라우저 호환성

Chrome, Firefox, Safari, Edge를 포함한 모든 최신 브라우저에서 작동합니다.

## 제한사항

- 데이터는 브라우저의 localStorage에 로컬로 저장됩니다
- 서버 측 동기화가 없습니다
- 다중 사용자 기능이 없는 단일 사용자 인터페이스

## 개발 노트

이 애플리케이션은 다음 현대 웹 개발 실무를 보여줍니다:
- HTML, CSS, JavaScript의 명확한 분리
- 반응형 디자인 원칙
- 로컬 데이터 저장
